## Data Source

All plots read from `analysis/phase1/results/*/aggregated_metrics.json`. To regenerate:

```bash
python scripts/generate_publication_plots.py \
  --results-dir analysis/phase1/results \
  --output-dir analysis/phase1/images

python scripts/generate_compact_figures.py
```

To copy plots into the paper's image directory:
```bash
cp analysis/phase1/images/*.png analysis/Articles/Publish/images/plots/phase1/
```
